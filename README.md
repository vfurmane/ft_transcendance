# ft_transcendence

A web Pong game with real-time multiplayer.

## Features

None.

## Getting started

### Prerequisities

You need the following programs to run the application:

- Git
- NodeJS
- NPM

### Installation

1. Clone the repository

```sh
git clone https://github.com/vfurmane/ft_transcendence
```

2. Install the dependencies

```sh
cd ft_transcendence
npm install
```

3. Start the server

```sh
npm start
```
